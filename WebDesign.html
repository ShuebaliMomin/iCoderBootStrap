What is Web Designer Exactly?

A web developer would be responsible for using various components like the engine, transmission, wheels, etc. to construct a fully-functional car that is error-free in its technical aspects. A useful way of illustrating the difference between a web developer and a web designer, is framing the comparison through something we all know. So, I want you to imagine a web developer and a web designer getting together to build a car.
